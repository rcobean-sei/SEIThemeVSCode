# Changelog

All notable changes to the SEI Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-14

### Added
- Initial release of SEI Theme
- SEI Dark theme variant
- SEI Light theme variant
- Complete color palette based on SEI Brand Guidelines Version 1.1
- WCAG AA accessible color combinations
- Comprehensive syntax highlighting for:
  - JavaScript/TypeScript
  - Python
  - JSON
  - CSS/SCSS
  - HTML
  - Markdown
  - And more!
- Semantic token color support
- Custom bracket pair colorization
- Git decoration colors
- Terminal color scheme
- Markdown-optimized colors for better readability

### Design Decisions
- Used CREAM (#FFFFF8) and BLACK (#141018) as base colors per brand guidelines
- RED 1 (#F01840) as primary accent color throughout both themes
- Purple tones (PURPLE 1-5) for depth and sophistication
- Reduced bold styling in dark theme for better readability
- Softened markdown colors (headings, lists) to reduce eye strain
- Maintained strong brand presence with strategic RED 1 usage

### Theme Features
- Clean, modern UI with layered backgrounds
- Subtle selection and highlight colors
- Vibrant but not overwhelming accent colors
- Consistent experience across light and dark themes
- Professional appearance suitable for client-facing presentations
